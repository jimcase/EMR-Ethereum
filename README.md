# EMR-Ethereum
[![Build Status](https://travis-ci.org/jimcase/EMR-Ethereum.svg?branch=main)](https://travis-ci.org/jimcase/EMR-Ethereum)

Electronic Medical Records PoC.



How to use:
```bash
❯ nvm use v14.6.0 
```
```bash
❯ npm install 
```
```bash
❯ truffle compile 
```
```bash
❯ ganache-cli 
```
```bash
❯ truffle test 
```
```bash
❯ truffle deploy
```

How to launch web app:
```bash
❯ react-scripts start
```
